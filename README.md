**怎么游玩:** 
1. 安装Node.js
2. 终端运行 `npm install`
3. 终端运行 `npm run dev`

**怎么构建android端** 
1. 下载Android Stduio (如果没有的话)
   "https://developer.android.google.cn/studio"
2. 命令行/终端输入  npx cap add android
3. 命令行/终端输入  npx cap sync android
   会自动跳转到Android Stduio
4. 点击Android Stduio 上面的三条横杠，然后点击
   build->Generate App Bubbles or APK->Generate APK 
   在构建完成后，右下角弹出的窗口点击"locate"即可
